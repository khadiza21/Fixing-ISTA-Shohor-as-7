Ans: Difference between Local Storage and Session Storage -
  1. Local Storage data is permanent in web storage API but Session storage data is short-term in web storage API.
  2. When the browser or tab is closed session storage data will be removed but local storage data doesn't remove until when will remove it manually.
  3. Local Storage size/capacity is generally 10 MB or 5 MB and Session storage size/capacity is 5MB.
  4. Session storage works for a definite browser tab cause short-term but Local Storage works for all browsers and tabs cause local storage data is permanent.
  5. Local storage has no expiration time but session storage has an expiration time when the browsing session will end.
